# Oceania Web Atlas

A curated directory of personal websites (blogs, portfolios, digital gardens) by people living in or from Oceania: Australia, Aotearoa/New Zealand, Papua New Guinea, Fiji, and other Pacific Islands: [zacharykai.net/lists/oceania](https://zacharykai.net/lists/oceania)

## Submission Guidelines

To be included in the Oceania Web Atlas:

- You must be from or living in Oceania
- It must be your personal site, blog, portfolio, or store
- The site should reflect something about yourself

## How To Submit Your Site

### Option 1: Pull Request

1. Fork this repository
2. Add your entry to `index.html` in alphabetical order by your name
3. Follow the existing format exactly
4. Submit a pull request with the title: "Add [Your Name] to directory"

**Entry Format:**

```html
<li>
    <strong><a href="https://yoursite.com">Your Name</a></strong>
    <ul>
        <li><strong>Location:</strong> Your City/Region, Country</li>
        <li><strong>Pronouns:</strong> your/pronouns (optional)</li>
    </ul>
</li>
```

**Important:**
- Use your actual name or online handle
- Include location (city/region optional, country required)
- Pronouns are optional but encouraged
- Alphabetize by first name

### Option 2: Email Submission

Email your submission to: **hi@zacharykai.net**

Include:
- Name (or handle)
- Homepage URL
- Location in Oceania
- Pronouns (optional)

## Guidelines For Contributors

- **Alphabetical Order:** Entries are sorted alphabetically by name
- **One Entry Per Person:** Only one site per person
- **Active Sites Only:** Your site should be actively maintained
- **Personal Sites Only:** No corporate sites, agencies, or promotional pages

## Review Process

All submissions are reviewed manually. I aim to review pull requests within 7 days. Submissions may be edited for clarity, grammar, or formatting to maintain consistency across the directory.

## Updates & Changes

To update your existing entry:
- Submit a pull request with the changes
- Or email: hi@zacharykai.net

To remove your entry:
- Submit a pull request removing your entry
- Or email: hi@zacharykai.net
